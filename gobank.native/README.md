# Go Bank

mobile bank app constuído em react-native

## Ferramentas usadas

- React Native
- expo
- typescript 
- [react navigation - stack](https://reactnavigation.org/)
- styled-components

## Local start

- terminal:
   1. yarn install
   2. yarn start
- inicie com expo go no seu smartphone/emulador
